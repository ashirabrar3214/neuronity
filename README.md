# Neuronity Frontend

The frontend application for Neuronity - an AI agent orchestration platform.

## Files

- `canvas.html` - Main canvas interface
- `canvas.js` - Canvas functionality
- `canvas.css` - Canvas styles
- `agent-training.js` - Agent training interface
- `agent-training.css` - Agent training styles
- `index.html` - Landing page
- `main.js` - Electron main process
- `renderer.js` - Electron renderer process
- `style.css` - Global styles

## Setup

```bash
npm install
npm start
```
